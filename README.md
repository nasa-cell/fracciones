# Fracciones Master

Fracciones Master es una app web interactiva para practicar operaciones con fracciones en modo juego. Incluye selección de operaciones, dificultad, modo con o sin alternativas y una interfaz visual con fondo de video.

## Características

- Suma, resta, multiplicación y división de fracciones
- Tres niveles de dificultad: fácil, medio y difícil
- Modo con alternativas o respuesta libre
- Sistema de vidas, errores, puntos y progreso
- Diseño responsive y animaciones sencillas

## Requisitos

Solo necesitas un navegador moderno.

## Uso

1. Abre [index.html](index.html) en tu navegador.
2. Selecciona las operaciones que quieras practicar.
3. Elige la dificultad y el modo.
4. Presiona "Comenzar".

## Estructura del proyecto

- [index.html](index.html): estructura de la interfaz
- [styles.css](styles.css): estilos y diseño visual
- [script.js](script.js): lógica del juego y generación de problemas
- [video fondo.mp4](video%20fondo.mp4): video de fondo
- [fracciones.mp4](fracciones.mp4): video de final
